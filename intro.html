Getting Started with HTML

In this tutorial, we'll learn the basics of HTML (Hypertext Markup Language), the foundational language for creating web pages.

What is HTML?

HTML stands for Hypertext Markup Language. It is the standard language used to create and structure web pages. HTML defines the content of web pages and allows the browser to display it.

Basic Structure of an HTML Page

Here’s the basic structure of an HTML document:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage!</p>
  </body>
</html>

The code above is the skeleton of a basic HTML page. Let’s break it down:

  • <!DOCTYPE html>: Declares the document type as HTML5.
  • <html>: The root element that contains all other elements on the page.
  • <head>: Contains meta-information about the page, like the title.
  • <body>: Contains the visible content of the page, like text and images.
  • <h1>: A header element for the main title.
  • <p>: A paragraph element for body text.

Conclusion

Now you’ve seen the basic structure of an HTML document and learned how to create a simple webpage. In the next tutorial, we will explore about HTML.
