Learn About Block and Inline Elements

Block Elements

Block elements take up the entire width of their parent container and start on a new line. Examples include <div>, <p>, and <h1>.

<div>
  <h1>Block Element</h1>
  <p>This is a block element</p>
</div>

Inline Elements

Inline elements take only as much width as they need and do not break the flow of content. Examples include <span>, <a>.

<p>This is a paragraph with an inline inline text element.</p>

Combining Block and Inline Elements

Block and inline elements can be used together to structure complex layouts. For example, a <div> can contain both block and inline elements like <span>.

<div>
  <p>This is a block-level element with an inline element inside.</p>
</div>
