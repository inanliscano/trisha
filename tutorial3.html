HTML Styles Tutorial

Learn about HTML styles, how to apply them, and their usage in web development.

Introduction to HTML Styles

HTML styles refer to the visual appearance and layout of elements on a webpage. You can style HTML elements using three main techniques:

  • Inline styles: Directly applied to HTML elements via the style attribute.
  • Internal styles: Defined inside the <style> tag within the <head> of the HTML document.
  • External styles: Defined in separate CSS files and linked using the <link> tag.

Inline Styles Example

Inline styles are applied directly within HTML tags using the style attribute. Here's an example:

<p style="color:red;">This is a red paragraph using inline style.</p>

The style here directly changes the color of the paragraph text to red.

Internal Styles Example

Internal styles are placed within the <style> tag in the HTML documentâ€™s <head> section:

<style>
  p {
    color: blue;
    font-size: 20px;
  }
</style>

This style block will make all <p> elements have blue text and a font size of 20px.

External Styles Example

External styles are written in separate CSS files and linked in the HTML document using the <link> tag. Here's an example:

<link rel="stylesheet" href="styles.css">

In this case, all styles are stored in an external file named styles.css and applied to the HTML page.
